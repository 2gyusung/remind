# 명령어
- pwd : 현재 파일 위치 경로 명령어
- mkdir class : 현재 위치 폴더 생성
- ls :  현재 위치 폴더명
- ls  -al : 숨겨진 파일 보이는 명령어
- cd .\class\ : 해댱하는 파열 경로 이동(tab 키 입력 시 자동입력)
- cd .. 현재 폴더 위치 경로에서 상위로 이동
- cp -R class class2 : 현재 폴더에 class2이름으로
-  rm -r class  해당하는 폴더 삭제하는 명령어
